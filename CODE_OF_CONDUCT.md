# Code of Conduct

cargo-target-gc aims to be a practical, respectful project for Rust developers.

Contributors and maintainers are expected to:

- Use welcoming, professional language.
- Focus criticism on code, design, documentation, and behavior.
- Assume good intent while still asking for clear evidence.
- Avoid harassment, insults, discriminatory language, and personal attacks.
- Respect maintainers' decisions about project scope and safety tradeoffs.

Unacceptable behavior can be reported privately to the maintainer. Maintainers
may remove comments, close issues, or block participants when needed to keep the
project productive.
