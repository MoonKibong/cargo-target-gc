# Localized Quick Starts

These quick starts summarize the safety-critical parts of cargo-target-gc in
multiple languages. The English README remains the source of truth for full
details, but these files ship with the crate so users can quickly understand:

- what cargo-target-gc scans;
- where to run `cargo target-gc`;
- when files can be deleted;
- how to configure retention.

## Contributor Rule

When changing user-facing docs, CLI command names, flags, configuration keys, or
safety behavior, update every localized file in this directory. If a change does
not affect translated docs, say so in the pull request.
